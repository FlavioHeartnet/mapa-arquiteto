# O Mapa do Arquiteto — Wiki de Estudos

Wiki construída a partir do board Miro **"O MAPA DO ARQUITETO"** (autor: **Renato Augusto**) — um roadmap de carreira em engenharia de software que vai da **Largada** (fundamentos) até virar **Arquiteto de Soluções**, passando pelas fases **Programador Júnior → Pleno/Sénior → Arquiteto**.

> **Fonte:** [board no Miro](https://miro.com/app/board/uXjVIrqt8ao=/) (protegido por senha).
> Cada tópico traz **explicação inline** (para você não precisar sair daqui), **vídeos** e **links de aprofundamento**. Aprofundamento maior na metade **Arquiteto (sênior)**.
> Na versão HTML (`index.html`), **clique em qualquer tópico** para abrir o painel de detalhes com tudo isso.

## Como usar

1. Comece por [00-roadmap.md](00-roadmap.md) para entender a sequência.
2. Estude fase por fase (01 → 02 → 03), marcando o que já domina.
3. Use [04-livros.md](04-livros.md) e [05-videos.md](05-videos.md) como material de apoio de cada tópico.
4. A versão HTML interativa (artifact) traz a mesma informação com navegação e links clicáveis.

## Marcos (checkpoints) do mapa

- 🏁 **Largada** — início
- 🏆 **Programador Júnior** — concluiu os fundamentos
- 🏆 **Programador Completo!** — concluiu Pleno/Sénior → vira **Arquiteto de Software**
- 🏆 **Arquiteto de Software → Arquiteto de Soluções**
- 🎽 **Parabéns!** — linha de chegada
